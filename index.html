A type of attack on web systems, which consists in injecting hacker code into the page displayed by the web system and interacting with the attacker's web server.

Types

  • Non-persistent.
  • Persistent.
  • DOM based XSS.

Threat Targets

  • Theft of cookies.
  • DoS attacks.
  • Attacks on the user's browser, data theft.
  • Performing any actions on the site under a user account.

How to prevent

  • Encoding of HTML control characters.
  • Secure validation of untrusted HTML input.
  • Cookie security.
  • Disable scripts.
  • Cookie option.
